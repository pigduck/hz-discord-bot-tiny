# HZ Discord Bot Tiny
您的 WooCommerce 智慧管理助手


## 介紹

專為電商店主設計，讓訂單管理變得前所未有的便捷！

* **即時訂單通知**：隨時隨地接收新訂單通知，讓您第一時間掌握商機。
* **訂單狀態快速更新**：只需簡單指令即可變更訂單狀態，提升管理效率。
* **快速訂單查詢**：使用便捷指令即可查詢訂單詳細資訊，節省您的寶貴時間。
* **商品查詢**：透過指令輸入商品名稱即可查詢商品詳細資訊，讓您隨時掌握商品資訊。
* **商品修改**：透過指令輸入即可修改商品價格、庫存等資訊，讓您輕鬆管理商品。


讓 HZ Discord Bot Tiny 為您的 WooCommerce 商店帶來智能管理的升級！
